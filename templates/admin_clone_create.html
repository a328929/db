本页做什么

先确认源群适合克隆，再创建一个空副本。创建成功后，去“继续克隆消息”页面继续。

  1. 1. 选择源群按消息量或最后更新时间找到要克隆的群组。
  2. 2. 开始检查读取数据库摘要，并确认第二账号是否能创建副本。
  3. 3. 填写副本信息确认标题、类型与操作确认码。
  4. 4. 创建空副本只创建结构，不迁移历史消息和媒体。
  5. 5. 继续迁移或查看详情从下方“最近创建记录”继续到迁移页，或直接打开记录详情页排查。
